## 설치 및 실행 방법
```
// 레포지토리 클론
git clone https://github.com/Derek-94/modu-todo-assignment.git
// 디렉토리 변경
cd modu-todo-assignment
// 종속성 설치
npm install
// 실행
npm start
```
